# Yandex Maps Nearby API

REST API для поиска ближайших объектов через Яндекс.Карты.

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone <ваш-репозиторий>
cd yandex-maps-nearby-api

## 🌍 API для поиска географических объектов

API ищет ближайшие географические объекты через Яндекс.Геокодер:
- **Улицы** - названия улиц
- **Города** - населенные пункты
- **Районы** - административные районы
- **Станции метро** - объекты метрополитена
- **Дома** - здания и строения
- **Достопримечательности** - известные места

### Пример запроса
GET /api/v1/nearby?lat=55.7558&lon=37.6173&type=улица&limit=5


### Поддерживаемые типы
- `улица` (street) — поиск улиц
- `город` (locality) — поиск городов
- `район` (district) — поиск районов
- `метро` (metro) — поиск станций метро
- `дом` (house) — поиск домов
- `достопримечательность` (house) — поиск достопримечательностей
